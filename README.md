## Getting Started
경비 문서 작성 시, 다양한 툴을 통하여 작성 후 캡쳐 및 복사/붙여넣기를 통하여 작성하는 것이 번거로워 개발하게 되었습니다.

다음 기능을 지원합니다.

✅ 표 만들기
✅ 영수증 첨부를 위한 템플릿

표는 M사의 Excel, G사의 Sheet 등 복사/붙여넣기 시 각기 다른 크기의 셀이 복사되므로 준비하였습니다.
